GETTY/IO Apply
===

Stack
---

Utilizei as seguintes tecnologias para desenvolver esta aplicação::

- NodeJS
- NPM
- Webpack
- Grommet
- ReactJS
- LESS
- ES 6 / Babel (Module Loading)
- Tests
  - Karma (Test Runner)
  - Mocha (Test Framework)
  - Chai (BDD/TSS assertion library)

Pré requisitos
---

NodeJS instalado

Scripts

Iniciar [aplicação](http://localhost:3000):

```bash
$ npm start
```
Executar testes:

```bash
$ npm run test
```
